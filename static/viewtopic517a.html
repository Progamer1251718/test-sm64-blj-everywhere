You can fix it with a parallel cable (aka printer cable or LPT cable) and a (probably old) PC that has a parallel port:

http://www.goldeneyeforever.com/gamesha ... torial.php

http://www.assemblergames.com/forums/sh ... -3-to-Work

After connecting it you need to reset the firmware using a special program.

Note that most parallel/USB adapters are a simpler type that won't work with GameShark, but there are SOME cables which are true parallel port emulators and support all the pins. Those should work, but they are probably hard to come by and a lot more expensive as well.
