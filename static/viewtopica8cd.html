People have different reasons for choosing to use a keyboard instead of a controller. Personally, I couldn't wait for getting a controller to start speedrunning, so I began using my keyboard, and now I'm used to it. I like the 'challenge'.

Hafus wrote:When you are spinning bowser with keyboard you can hold down left and keep pressing right and he will spin :)

This is one of the very few keyboard only tricks. Different keybindings / emulators have variations on this which allow you to do it (in my case, I hold right and tap left). It's still good to know how to spin the 'proper' way though.

Playing using a keyboard is not as hard as people think. It mainly takes getting used to, with a few tricks to learn, just like everything else. I guess this is an indication that I could consider making a guide.
