I found this a few weeks back and I was suggested to post this here. So here is my log skip time saver in TTM. I am not sure how much time it would save compared to using the log but, I would guess maybe around 3 seconds but, this could be used multiple times in a run to save more time. Warning: Audio is kinda loud.

http://www.youtube.com/watch?v=JqiGr0IdCUw

You obviously do not need to use my camera setup that just makes it easier. This has a backup strat and a backup strat for the backup strat. If you do not get enough distance and hit the wall you can just do a 2nd/3rd frame wallkick and drift back to grab the ledge. If you try and do the backup strat and get a first frame you can drift and dive to get to the log to recover.

I am interested in seeing people's thoughts on this time save and optimizing it if possible. :)
